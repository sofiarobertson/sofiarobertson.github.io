Research


  • A Real-Time Web-Based Dashboard for Comprehensive Green Bank Telescope System Monitoring

    National Radio Astronomy Observatory/Green Bank Observatory

    Developed an integrated, real-time, web-based dashboard as a proof of concept with my mentor, Thomas Chamberlin. Dashboard aims to unify access to various Green Bank Telescope data sources and software systems within a single interface. Utilizing Django, a Python web framework, alongside the Chalice Client for telescope interaction, we retrieved the necessary information from the telescope and displayed it on the website using HTML templates along with CSS, Bootstrap, Matplotlib, and astropy.


  • Stability of Planetary Orbital Configurations Klassifier (SPOCK) and its Improvements

    Harvey Mudd College

    Worked on improving SPOCK, a machine learning model that predicts the orbital stability of planetary systems. Focused on exploring how N-body resonances can affect whether a system stays stable, with the goal of making the model's classifications easier to understand. Explorations led to implementing a Hill Factor feature, which measures how close a pair of planets is to their critical separation for long-term stability, providing a stability diagnostic for a two-body system. Enhanced existing features to improve its predictions and built stability tree visuals to determine how well the model performs.
